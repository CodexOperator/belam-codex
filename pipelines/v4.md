# Implementation Pipeline: V4 — Differential Output & Network Scale Exploration

## Description
Systematic exploration of network scale (nano → medium) with differential opponent-coded output and multiple encoding/decoding schemes. 32 SNN experiments × 2 output decoding schemes (linear + exponential differential) + 6 sklearn baselines across 3 walk-forward folds. Both pipeline phases live in a **single combined notebook** (`v4_combined.ipynb`) for side-by-side deep analysis.

## Key Innovations
- **Differential output (linear):** Two opponent neurons (UP/DOWN), `learnable_scale × (r_up − r_down)` → sigmoid — Phase 1 decoding
- **Differential output (exponential):** `sigmoid(exp(r_up) − exp(r_down))` — Phase 2 decoding, less punishing for near-misses
- Nano-scale networks (5–15 neurons) — minimal viable SNN probing, present in **both** phases
- Equilibrium encoding with sequential eval (persistent membrane state across candles)
- Scale × encoding × decoding factorial design

**Both pipeline phases are sections of a single notebook.** Do not produce two separate notebooks. The combined notebook enables direct cell-by-cell comparison and a unified cross-phase analysis section.

### Decoding Scheme Implementation Note

The `SNNDifferentialV4` class accepts `decoding_scheme: str` parameter:
- `'linear'`: `scaled_diff = learnable_scale * (r_up - r_down)` → `BCEWithLogitsLoss(scaled_diff, target)`
- `'exponential'`: `prob = sigmoid(exp(r_up) - exp(r_down))` → `BCELoss(prob, target)`

For magnitude mode (`output_mode='magnitude'`): decoding_scheme is ignored — both phases use `HuberLoss(predicted_return, actual_return, delta=0.01)`.

### Nano Network Confirmation (Phase 2)

**Yes — nano networks (5, 9, 15 neurons) are explicitly in Phase 2.** They are in both phases:
- Phase 1 sections 5.3–5.5 (linear decoding)
- Phase 2 sections 6.3–6.5 (exponential decoding)

The nano × exponential combination is a primary research question: does `sigmoid(exp(r_up) − exp(r_down))` help tiny networks that might otherwise produce near-threshold, ambiguous predictions?
