pub unsafe extern "C" fn get_jwt(
    connection_id_ptr: *const u8,
    bytes_source_id: *mut BytesSource,
) -> u16
Expand description

Finds the JWT payload associated with connection_id. A [ByteSourceId] for the payload will be written to target_ptr. If nothing is found for the connection, [ByteSourceId::INVALID] (zero) is written to target_ptr.

This must be called inside a transaction (because it reads from a system table).

§Errors

Returns an error:

  • NOT_IN_TRANSACTION, when called outside of a transaction.

§Traps

Traps if:

  • connection_id does not point to a valid little-endian ConnectionId.
  • target_ptr is NULL or target_ptr[..size_of::<u32>()] is not in bounds of WASM memory.
  • The ByteSourceId to be written to target_ptr would overflow u32::MAX.
