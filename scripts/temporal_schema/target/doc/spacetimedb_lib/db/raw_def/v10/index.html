ABI Version 10 of the raw module definitions.

This is a refactored version of V9 with a section-based structure. V10 moves schedules, lifecycle reducers, and default values out of their V9 locations into dedicated sections for cleaner organization. It allows easier future extensibility to add new kinds of definitions.
