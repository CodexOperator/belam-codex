Accumulate agent context using RFC 7396 JSON Merge Patch semantics. FLAG-4 fix: well-defined merge behavior.

Merge rules (JSON Merge Patch / RFC 7396):

  • Object keys: recursively merge nested objects
  • Arrays: concatenate (extension to RFC 7396 for our use case — we want to ACCUMULATE decisions/flags/questions, not replace them)
  • Null values: remove the key
  • Primitive values: new overwrites old
