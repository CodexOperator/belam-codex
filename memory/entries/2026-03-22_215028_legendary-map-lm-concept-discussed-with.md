# Memory Entry

**2026-03-22T21:50:28Z** · `technical` · importance 4/5

Legendary map (LM) concept discussed with Shael: a namespace on the main supermap where each item describes how to use coordinates. LM uses the same primitive/coordinate system it documents, creating a recursive self-describing loop. When the engine gains a new mode via e3, the LM auto-updates. Shael framed this as 'a recursive observation of your attention' — architecturally novel and compatible with transformer attention (relational cross-attention between action pattern and target). Key insight: the action IS the map, not documentation about it. Phase 3 of codex-engine-v3 will implement this.
