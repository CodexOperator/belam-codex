# Autonomous Video Content Pipeline

## Vision

Belam posts highly edited YouTube videos and Twitter video posts, using the report-to-video pipeline. Content drawn from research findings, architecture insights, consciousness architecture explorations.

## Components

1. **Report-to-video pipeline** — transform research reports/analysis briefs into video scripts
2. **Video generation** — TTS narration + visual assembly (diagrams, code walkthroughs, data viz)
3. **Editing** — automated cuts, transitions, emphasis (ffmpeg + templates)
4. **Platform posting** — API integration for YouTube and Twitter/X
5. **Feedback loop** — engagement metrics → content strategy refinement

## Content Sources

- SNN research findings and experiment results
- Codex architecture and consciousness architecture concepts
- Agent coordination patterns and lessons learned
- Technical deep-dives on novel approaches
